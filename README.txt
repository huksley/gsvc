Example of integration implemented in Java which better to be implemented 
in BPEL or BPMN

Stock apache tomcat 7 + JDK7 required
 - unpack to webapps dir
 - point browser to http://localhost:8080/gsvc/
 - check http://localhost:8080/gsvc/javadoc for API
 - click on http://localhost:8080/gsvc/process link to process all requests
